## 🎨 Comment utiliser le prompt amélioré

### **ÉTAPE 1: Copier le prompt**
1. Ouvrez le fichier `FIGMA_MAKE_PROMPT.md`
2. Copiez le contenu du bloc ```...```

### **ÉTAPE 2: Aller sur Figma Make**
1. Allez sur https://www.figma.com/ai/create
2. Collez le prompt dans le champ "Description"
3. Laissez les paramètres par défaut

### **ÉTAPE 3: Générer les designs**
1. Cliquez sur **"Generate"** ou **"Créer"**
2. Attendez 2-5 minutes que Figma génère tous les écrans
3. Explorez les résultats

### **ÉTAPE 4: Affiner les designs**
- Si un écran ne vous plaît pas: regenerate cet écran
- Utilisez l'éditeur Figma pour polir les designs
- Exportez en PNG/JPG pour votre présentation
